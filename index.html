xIDS: Explainable Intrusion Detection System
M.S. Capstone + Portfolio Project, GVSU
Complete

An AI-powered network intrusion detection system that does not just flag attacks, it explains why. Tiered detection pipeline using Isolation Forest and XGBoost on CICIDS2017 traffic data.
